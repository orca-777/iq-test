# 管培生综合评估系统 - 使用说明

## 快速启动

双击运行 `start.bat`，启动后控制台会显示所有访问地址。

---

## 访问地址

### 本地运行（电脑直接访问）

| 页面 | 地址 |
|------|------|
| 考生答题入口 | http://localhost:3000 |
| 管理后台 | http://localhost:3000/admin |

### 局域网访问（同一 WiFi 下其他设备）

| 页面 | 地址 |
|------|------|
| 考生答题入口 | http://[你的电脑IP]:3000 |
| 管理后台 | http://[你的电脑IP]:3000/admin |

> 查看本机IP：打开命令提示符，输入 `ipconfig`，找到"IPv4 地址"

### GitHub Pages 公网入口（需配置后端地址）

1. 考生打开：https://serena-11a.github.io/iq-test/
2. 首次访问会提示输入后端地址，填入：`http://[你的电脑IP]:3000`
3. 或直接发给考生带参数的链接：  
   `https://serena-11a.github.io/iq-test/?api=http://[你的电脑IP]:3000`

> ⚠️ 使用 GitHub Pages 入口时，考生电脑必须与服务器在同一局域网内。

## 数据存储

- **主存储**：飞书多维表格（实时同步）
- **本地备份**：`server/data/results_backup.json`（飞书失败时自动备份）
- **题库**：`server/data/assessment.db`（SQLite数据库）

## 常见问题

**Q: 服务器启动失败？**  
A: 检查 3000 端口是否被占用，运行 `netstat -ano | findstr :3000`，若有占用，在任务管理器结束对应进程。

**Q: 飞书数据同步失败？**  
A: 数据会自动保存到本地 `server/data/results_backup.json`，不会丢失。可在管理后台导出 CSV。

**Q: 忘记管理员密码？**  
A: 删除 `server/data/assessment.db` 后重新运行 `start.bat` 初始化（⚠️ 会清空题库，需重新导入）。

**Q: 如何修改题目？**  
A: 登录管理后台 → 题库管理 → 点击编辑按钮修改对应题目。
